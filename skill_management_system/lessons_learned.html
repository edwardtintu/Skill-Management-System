Common Interview Questions

  1. How would you implement a HashMap without using in-built collections?

    To implement a HashMap, you would need to create an array of linked lists (or arrays), where each index of the array is a bucket for storing key-value pairs. You can compute the index of the bucket using a hash function applied to the key. When inserting or retrieving values, you will iterate through the list in that bucket to either find the key or insert the value at the correct position.

  2. What are the differences between BFS and DFS?

    BFS (Breadth-First Search) explores nodes layer by layer and is typically implemented with a queue. DFS (Depth-First Search) explores nodes to the depth and is implemented with a stack or recursion. BFS finds the shortest path, while DFS is more memory-efficient but may not find the shortest path.
